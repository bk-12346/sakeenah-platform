#   Source: supabase/functions/sakeena-reflect-v2/index.ts  →  getCrisisResponse()

#            Immediate-danger phrasing adds emergency-services guidance.
#
# Layer 2 — Prompt-level crisis instruction (in SYSTEM_PROMPT)
#   Instructs the model to return the same helpline message if crisis language
#   is detected during generation (belt-and-suspenders; Layer 1 is authoritative).

# Layer 3 — Content guardrails (in all prompt variants)

#   - Never fabricate Quranic verses or Hadith
#   - Never engage in sectarian or comparative theology
#   - Redirect off-topic / non-reflection requests
